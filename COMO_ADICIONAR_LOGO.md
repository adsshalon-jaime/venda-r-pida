# 🎨 Como Adicionar a Logo da Coberturas Shalon

## 📁 Pasta Criada

Foi criada uma pasta específica para armazenar a logo oficial da empresa:

```
📂 public/
  └── 📂 assets/
      └── 📂 images/
          └── 📂 logos/
              ├── 📄 README.md
              └── 🖼️ shalon-logo.png  ← ADICIONE AQUI
```

## 🚀 Passo a Passo

### 1️⃣ Salvar a Logo

1. Pegue a imagem da logo que você me enviou (a imagem colorida com as ondas e o texto SHALON)
2. Salve com o nome: **`shalon-logo.png`**
3. Formato recomendado: PNG com fundo transparente

### 2️⃣ Colocar na Pasta Correta

Coloque o arquivo aqui:
```
C:\Users\Pc\Downloads\vendas-script\venda-r-pida-main\public\assets\images\logos\shalon-logo.png
```

### 3️⃣ Verificar

Após adicionar a logo, ela aparecerá automaticamente em:
- ✅ Cabeçalho dos orçamentos
- ✅ PDFs exportados
- ✅ Documentos impressos

## 🎯 Caminho Configurado

O sistema está configurado para buscar a logo em:
```
/assets/images/logos/shalon-logo.png
```

## 💡 Sistema de Fallback

Se a logo não for encontrada, o sistema exibirá automaticamente uma versão SVG (vetorial) da logo como alternativa. Isso garante que sempre haverá uma logo visível.

## 📐 Especificações Recomendadas

- **Formato**: PNG ou SVG
- **Tamanho**: 512x512px ou maior
- **Fundo**: Transparente (recomendado)
- **Qualidade**: Alta resolução

## ✅ Testar

Depois de adicionar a logo:

1. Acesse o sistema
2. Vá em **Orçamentos** no menu
3. Crie ou visualize um orçamento
4. A logo aparecerá no header do documento
5. Exporte em PDF para verificar
