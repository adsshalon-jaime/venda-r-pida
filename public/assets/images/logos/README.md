## 📋 Instruções para Adicionar a Logo

### Opção 1: PNG (Recomendado para começar)
1. Salve a logo como: `shalon-logo.png`
2. Coloque nesta pasta: `public/assets/images/logos/`
3. Tamanho recomendado: 512x512px ou maior
4. Formato: PNG com fundo transparente

### Opção 2: SVG (Melhor qualidade)
1. Salve a logo como: `shalon-logo.svg`
2. Coloque nesta pasta: `public/assets/images/logos/`
3. Formato vetorial para máxima qualidade

## 🔗 Onde a Logo é Usada

A logo é exibida automaticamente em:
- ✅ Documentos de orçamento (header)
- ✅ PDFs exportados
- ✅ Impressões de orçamentos

## 📐 Especificações Técnicas

**Cores da Marca:**
- Amarelo: #FFD700
- Vermelho: #EF4444
- Azul: #3B82F6
- Roxo/Azul escuro: #4F46E5

**Dimensões:**
- Mínimo: 256x256px
- Recomendado: 512x512px ou 1024x1024px
- Formato: Quadrado ou retangular

## 🚀 Como Testar

Após adicionar a logo:
1. Acesse a página de Orçamentos
2. Crie ou visualize um orçamento
3. A logo aparecerá no header do documento
4. Exporte em PDF para verificar a qualidade

## 💡 Fallback

Se a logo não for encontrada, o sistema exibirá automaticamente uma versão SVG incorporada no código como alternativa.
